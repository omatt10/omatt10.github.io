CSCE242: Client-Server Computing

GitHub Repository

Class Description: CSCE242 Web Application and Design. Students will utilize HTML and CSS to design the front-end of a website; matching a given specification. Secondly, they will use JavaScript to add functionality to their website. Lastly, they will learn to utilize node and server side JavaScript to store and retrieve data from a server.
